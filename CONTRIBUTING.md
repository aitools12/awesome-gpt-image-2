# Contributing

This repository accepts additions for GPT Image 2 prompts and examples, but only when the source path is clear.

## Submission Rules

- Include a visible upstream source.
- Include the original post, page, or repository link.
- Keep the local image filename descriptive and stable.
- Add the matching record to `data/gpt-image-2-dev-cases.json`.
- Do not submit examples without attribution.

## Preferred Categories

- UI mockups
- Infographics
- Posters
- Product ads
- Character sheets
- Social screenshot style compositions

## Suggested Workflow

1. Copy the image into `assets/reference/`.
2. Add source metadata to `data/gpt-image-2-dev-cases.json`.
3. Place the case in the most relevant docs page under `docs/`.
4. Update `README.md` and `README_zh-CN.md` if the case is strong enough for the main page.
