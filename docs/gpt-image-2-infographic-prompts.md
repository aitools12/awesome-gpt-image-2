# GPT Image 2 Infographic Prompts and Educational Examples

This page groups GPT Image 2 prompts for infographics, encyclopedia cards, museum-style explainers, flowcharts, and structured educational layouts.

## Recommended GPT Image 2 Infographic Prompt Patterns

- Define the knowledge format first: infographic card, anatomy chart, process diagram, travel guide, or museum wall label.
- Ask for visible structure such as sections, labels, callouts, legends, and comparison blocks.
- Mention the editorial tone you want: educational, museum-grade, textbook-like, collectible card, or clean science poster.
