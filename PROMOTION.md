# Promotion Plan for Awesome GPT Image 2 Dev

## Positioning

This repository is not just a site mirror. It should be promoted as:

- a public prompt and showcase resource for `gpt-image-2.dev`
- a GitHub discovery layer for people searching `GPT Image 2`
- a reusable prompt collection for image generation and editing workflows
