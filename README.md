## What This Repository Is

`gpt-image-2.dev` is positioned as an AI image generator and browser-based image editor built around the `GPT-Image-2` model. The public site emphasizes:

- text-to-image generation
- image editing with natural-language instructions
- reference-based remix workflows
- product and e-commerce image creation
- creator and marketing-friendly output

This repository does not try to mirror the private product internals. It focuses on public discovery and reusable prompt knowledge:

- public showcase snapshots
- categorized prompt cases
- JSON starter datasets
- English and Chinese landing-style documentation
- promotion copy for future distribution

## How to Extend This Repo for Other Sites

If you want to publish more website-oriented AI repos under `aitools12`, reuse this same pattern:

1. Keep the repository name in `awesome-<site>` or `awesome-<tool>` style.
2. Add one public banner under `assets/`.
3. Add two README files at minimum: English and Simplified Chinese.
4. Group cases by use case, not by random chronology.
5. Maintain one prompt dataset and one case dataset.
6. Keep official links near the top of the repo.
7. Add promotion docs on day one rather than as an afterthought.
