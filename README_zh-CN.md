## 这个仓库是干什么的

`gpt-image-2.dev` 当前对外呈现的核心能力主要是：

- 文生图
- 图片编辑
- 参考图重混
- 电商产品图生成
- 创作者和营销类图片制作

这个仓库不做后台代码镜像，重点做公开资料层：

- 公开展示图引用
- 场景化 prompt 模板
- 按类别整理的案例块
- 结构化 JSON 数据
- 中英文 README
- 推广和投放文案
